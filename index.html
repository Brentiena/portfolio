Google Sheets koppelen — stap voor stap

1
Ga naar sheets.google.com en maak een nieuw spreadsheet aan. Noem het bijvoorbeeld Portfolio.
2
Maak in rij 1 de volgende kolomkoppen aan (exact zo, inclusief hoofdletters):

Naam · Type · Belegd · Startdatum · Maandelijks · ExpReturn · TotaleWaarde

Type = een getal: 0=ETF/Aandeel, 1=Crypto, 2=Obligatie, 3=Goud/Grondstof, 4=Cash, 5=Overig
Belegd = eenmalige startinleg (0 als je alleen maandelijks inlegt)
Startdatum = datum waarop je begon in te leggen, bijv. 2024-01-01
Maandelijks = maandelijks bedrag dat je bijlegt (0 als niet van toepassing)
TotaleWaarde = huidige waarde van je positie — update dit één keer per maand vanuit je broker app
3
Vul per positie een rij in. De kolom TotaleWaarde is de huidige waarde vanuit je broker — dit update je één keer per maand handmatig. Laat cellen leeg als je de waarde niet hebt, het dashboard valt dan terug op het berekende totaal belegd.
4
Publiceer het sheet als CSV: klik op Bestand → Delen → Publiceren op het web. Kies het juiste tabblad en stel het formaat in op CSV. Klik op Publiceren en kopieer de URL.
5
Plak de gekopieerde URL in het veld rechtsboven in het dashboard en klik op Sync ↻. Klik daarna op Opslaan zodat de URL onthouden wordt. Het dashboard laadt nu automatisch je live data.
6
De LiveKoers in Google Sheets wordt automatisch bijgewerkt door Google Finance. Klik in het dashboard op Sync ↻ om de nieuwste koersen op te halen.

GitHub Pages — eenmalige setup

1
Ga naar github.com en maak een gratis account aan als je die nog niet hebt.
2
Maak een nieuw repository aan. Noem het portfolio en stel het in als Private — zo is je code niet zichtbaar voor anderen, maar GitHub Pages werkt gewoon.
3
Upload dit index.html bestand naar het repository (klik op Add file → Upload files).
4
Ga naar Settings → Pages en stel onder Branch in: main / root. Klik op Save. Na een minuut is je dashboard live op https://jouwgebruikersnaam.github.io/portfolio
